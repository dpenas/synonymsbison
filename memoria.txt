El sistema se compila con el Makefile (previamente se eliminan todos los archivos innecesarios de la carpeta donde realizamos el make). Ejecutando la práctica (./synonyms) saldrá el menú donde podremos elegir entre buscar un sinónimo (solamente disponible en inglés) o la traducción de la palabra (disponible de inglés a español y de español a inglés). El siguiente pasó será incluir la palabra y, si existe, no dará ningún error (en caso de que no exista se nos avisará de ello). Adicionalmente para las traducciones hay que indicar la lengua de la propia palabra y aquella a la que queramos traducir.
Una vez hecho esto es sistema descargará dicha el HTML de dicha palabra, parseará el fichero de texto en busca de las traducciones, diferentes sentidos, etc. y creará una simple web en HTML donde se mostrará el resultado destacando con diferentes sintáxis los diferentes tipos de textos devueltos (por ejemplo, si buscamos un sinónimo de "cat", por una parte tendremos dichos sinóminos, que iran con un formato, y por otra parte los diferentes sentidos que puede tener gato: animal doméstico, herramienta de trabajo, etc. que se resaltarán en negrita, por lo que el sistema es capaz de detectar lo que verdaderamente devuelve el HTML). El sistema es similar para las traducciones.
